# Leleka Chorna — односторінковий сайт

Готовий статичний проєкт для GitHub Pages / будь-якого хостингу.

## Що всередині
- `index.html` — контент сторінки (Головна → Про мене → Формати і ціни → Як працюємо → Запис)
- `styles.css` — стилі (адаптивно, темна тема, акцентний зелений)
- `script.js` — мобільне меню, анімація появи блоків, поточний рік
- `assets/logo.svg` — мінімалістичний логотип (тимчасовий, замініть на власний)
- `assets/favicon.svg` — фавікон
- `assets/teacher-placeholder.jpg` — плейсхолдер для фото викладачки
- `assets/og-cover.jpg` — OG-зображення для прев’ю посилання

## Швидкий старт (GitHub Pages)
1. Створіть публічний репозиторій, наприклад `lelekachorna-site`.
2. Завантажте всі файли з цієї папки в корінь репозиторію.
3. У налаштуваннях репозиторію відкрийте **Pages** → **Deploy from a branch** → `main` / root.
4. Через кілька хвилин сайт буде доступний за посиланням формату `https://<ваш_логін>.github.io/<repo>/`.

## Налаштуйте під себе
- Замініть `assets/logo.svg` та `assets/teacher-placeholder.jpg` на реальні логотип і фото.
- У формі замініть email за потреби (атрибут `action` у `<form>` використовується сервісом FormSubmit).
- Посилання на соцмережі: Telegram та Instagram — вже підставлено.
- Тексти та блоки — редагуйте у `index.html`.

## Форми без бекенду
Форма відправляє дані через [FormSubmit](https://formsubmit.co/). За бажанням можете підключити інший сервіс (Formspree/Getform) — достатньо змінити `action`.
